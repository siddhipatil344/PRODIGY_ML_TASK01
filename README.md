# 🏠 House Price Prediction

## 📌 Project Description
This project is part of my Machine Learning Internship (Task 1).  
The goal is to predict house prices using a simple Linear Regression model.

## 📊 Dataset
The dataset contains housing information such as:
- Living Area (GrLivArea)
- Number of Bedrooms (BedroomAbvGr)
- Number of Bathrooms (FullBath)
- Sale Price (target variable)

## ⚙️ Steps Performed
1. Loaded dataset using Pandas  
2. Selected important features  
3. Split data into training and testing sets  
4. Trained Linear Regression model  
5. Predicted house prices  

## 🔧 Technologies Used
- Python  
- Pandas  
- Scikit-learn
